# 成果物説明: Personal Blog & Portfolio (DevLog)

就職活動や実績説明の際に活用いただける、本プロジェクトの技術的な詳細とアピールポイントのまとめです。

## 1. プロジェクト概要
自身のフロントエンド技術の知見やAI開発に関する記録を発信するための、モダンな技術スタックを用いたブログ・ポートフォリオ統合サイトです。

## 2. 使用技術（技術スタック）
- **Frontend**: React (v19), TypeScript
- **Styling**: Tailwind CSS (v4)
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: GitHub Pages (GitHub Actionsによる自動CI/CD)

## 3. 実装した主な機能
- **動的なコンテンツフィルタリング**: カテゴリ選択による記事のリアルタイム抽出。
- **全文検索機能**: 記事のタイトルや概要を対象としたインクリメンタルサーチ。
- **シングルページアプリケーション(SPA)体験**: `useState`によるビューの切り替えを用いた、スムーズな記事詳細表示。
- **レスポンシブデザイン**: モバイル・タブレット・PCすべてのデバイスで最適化された美しいレイアウト。
- **モダンなUI/UX**: グラスモルフィズム、グラデーション、インタラクティブなホバーアニメーションの採用。

## 4. 技術的なこだわり・工夫点
- **パフォーマンス**: Viteによる高速なビルドと、Tailwind v4の新しいエンジンの活用によるCSSサイズの最適化。
- **コードの品質**: TypeScriptによる厳格な型定義を行い、保守性と可読性を高めています。
- **ユーザー体験**: 記事一覧から詳細への遷移をコンポーネントレベルで制御し、ローディングを感じさせないシームレスな操作感を実現。
- **拡張性**: 記事データ（JSON形式）を追加するだけで、自動的にUIが更新されるデータ駆動型の設計。

## 5. 今後の拡張予定
- **Markdownパーサーの導入**: 外部ファイルからのMarkdown読み込み機能の追加。
- **Dark Mode対応**: ユーザーの好みに合わせたテーマ切り替え機能。
- **AIアシスタント機能**: 記事の内容を要約したり、関連する質問に答えるAIチャットの統合。
